# Engineering Quality Index (EQI) — Launch Page

A single-page announcement for the Mindbowser EQI Dashboard, built with the Mindbowser brand
palette (purple/teal/navy) and live category scores from the Epico EQI dashboard.

## View it live

Once GitHub Pages is enabled (see below), this page will be available at:

```
https://<your-github-username>.github.io/<repo-name>/
```

## Files

- `index.html` — the full page (HTML/CSS/JS, no build step needed)

## Updating

Edit `index.html` directly and push to `main` — GitHub Pages rebuilds automatically within
a minute or two.
